## Test Guide
### How to run tests
#### Test TCP
in the terminal, in the piman folder, run:
```
sudo python3 tcp/test_tcp.py
```

#### Test TFTP
in the terminal, in the piman folder, run:
```
sudo ./tftp_test.sh
```

#### Test DHCP
in the terminal, in the piman folder, run:
```
sudo ./dhcp_test.sh
```

#### Test Make_zipapp in piman
in the terminal, in the piman folder, run:
```
python3 test_piman.py
```
